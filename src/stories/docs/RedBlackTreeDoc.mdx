# Red Black Tree

It organizes numbers in a self balancing binary search tree and exposes various styling options.

## Example

```js
const Component = () => {
  const { ref, insert } = useRedBlackTree()

  const [insertValue, setInsertValue] = useState(0)

  return (
    <>
      <div>
        <input
          type="number"
          onChange={(elem) =>
            setInsertValue(parseInt(elem.currentTarget.value))
          }
        />
        <button onClick={() => insert(insertValue)}>Insert</button>
      </div>
      <RedBlackTree ref={ref} data={[2, 1, 3]} />
    </>
  )
}
```

## useRedBlackTree

This hook allows you to interact with your tree. Insert, remove, search and so much more!

It returns an object with the following properties.

<Table data={AVL_HOOK_PROPERTIES} />

##### `TraversalOrderType = 'inorder' | 'postorder' | 'preorder'`

##### `BinaryTreeCheckType = 'balanced' | 'complete' | 'perfect' | 'full' `
