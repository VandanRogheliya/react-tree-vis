# Max Heap

It organizes numbers in a max heap and exposes various styling options.

## Example

```js
const Component = () => {
  const { ref, insert } = useHeap()

  const [insertValue, setInsertValue] = useState(0)

  return (
    <>
      <div>
        <input
          type="number"
          onChange={(elem) =>
            setInsertValue(parseInt(elem.currentTarget.value))
          }
        />
        <button onClick={() => insert(insertValue)}>Insert</button>
      </div>
      <MaxHeap ref={ref} data={[2, 1, 3]} />
    </>
  )
}
```
