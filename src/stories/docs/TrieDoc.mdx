# Trie

It organizes words in a trie and exposes various styling options.

## Example

```js
const Component = () => {
  const { ref, insert } = useTrie()

  const [insertValue, setInsertValue] = useState('')

  return (
    <>
      <div>
        <input
          type="text"
          onChange={(elem) => setInsertValue(elem.currentTarget.value)}
        />
        <button onClick={() => insert(insertValue)}>Insert</button>
      </div>
      <Trie ref={ref} data={['cook', 'cold', 'cap', 'cooker', 'cool']} />
    </>
  )
}
```
